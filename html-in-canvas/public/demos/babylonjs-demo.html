This demo requires a browser with HTML-in-Canvas support (e.g. Chrome 150+ with the "Experimental Web Platform Features" flag enabled).

Rotation Mode

Switch modes between "continuous" or "every 2 seconds" rotation to see how the canvas re-ndering is handled in this demo. The canvas is only updated when there's a plane rotation OR when any children elements of the HTML texture change. See the console logs.
